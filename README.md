# Building a CPU from Scratch
### Seasons of Code 2026 — Web and Coding Club, IIT Bombay

Design and implement three generations of a processor in Verilog —
single-cycle, multi-cycle, and pipelined — each executing a bubble
sort algorithm on a real array in memory.

---

## Roadmap

| Week | Topic |
|------|-------|
| 1 | Verilog Basics |
| 2 | Circuit Design — ALU, Register File, MUX, PC |
| 3 | FSMs |
| 4 | CPU Architecture + Assembly |
| 5 | Single-Cycle CPU |
| 6 | Multi-Cycle CPU |
| 7 | Learn Pipelining  |
| 8 | Pipelined CPU |

---

## Checkpoints

- **Week 3** — All Verilog modules working with passing testbenches
- **Week 5** — Single-cycle CPU correctly sorts an array in simulation
- **Week 8** — Pipelined CPU hazard-free + final performance report
